## Customer App

Use this app for passengers.

1. Open the customer QR/link.
2. Enter name, phone, email optional, PNR, train, coach and seat.
3. Choose meal type.
4. Select food quantity.
5. Choose delivery slot.
6. Place order.
7. Open Track tab with the same phone number to see order status.
8. Use Customer Invoice button after order is active.
9. Use Feedback tab for feedback or complaint.
10. Use Track tab to see RK response, reply back and rate RK response.

Meal delivery timing:

- Breakfast: 07:00 to 10:00.
- Lunch: 12:00 to 15:00.
- Dinner: 19:30 to 22:00.
- Other items: 1-hour slots from 06:00 to 23:00.

## Command Center

Use this app for train manager, supervisors and admin users.

1. Open command center link.
2. Login if using internal credentials.
3. Watch badges on Customer Orders and Feedback tabs.
4. Customer Orders:
   - Accept new order.
   - Mark preparing.
   - Initiate delivery with delivery person name and phone.
   - Mark delivered.
   - Cancel only before delivered if needed.
5. Feedback:
   - Read feedback/complaint details.
   - See proof image/audio/video if uploaded.
   - Enter manager response.
   - Change status.
   - Save status so customer can see update.
6. Base Kitchen Indent:
   - Choose train and base kitchen.
   - Add items and quantity.
   - Send indent.
   - Watch fill rate and shortage lines.
7. Delivery Invoice:
   - Print invoice PDF.
   - Accept delivery.
   - Raise dispute with note/photo if mismatch.
8. Purchases / Store:
   - Record cash purchase or store inward.
   - Add catalog item when item should be visible to customer.
9. Staff Sale:
   - Enter salesperson sale after stock returns to pantry.
   - System deducts sold stock.
10. Masters:
   - Add/edit/hide food items.
   - Add/search train master.
   - Add/search base kitchen.
   - Bulk upload train/base kitchen master.
11. Super Admin:
   - Download backup.
   - Download food master.
   - Delete master only during testing.

## Base Kitchen App

Use this app for base kitchen team.

1. Open base kitchen link.
2. Select base kitchen and press Start.
3. Incoming Indents:
   - Review requested items.
   - Edit approved quantity if item is not available.
   - Save quantity.
   - Accept/process indent.
   - Create invoice.
4. Invoices:
   - View invoices sent to train.
   - Print invoice PDF.
5. Disputes:
   - Review dispute note and proof.
   - Edit final quantity/cost if needed.
   - Enter settlement note.
   - Resend final invoice to command center.
